# Example: Project Playbook

## User request

Design a practical workflow for producing short-form social media posts every week.

## Expected PavedPath behavior

Use Project Playbook. Gather or request enough context for platform, cadence, team size, and content type. If live research is available, prefer current creator workflows and platform docs. Produce an executable operating system.

Expected sections:

```text
Goal and constraints
Evidence-backed patterns
Recommended weekly workflow
Tool stack
Content pipeline steps
Review and publishing checklist
Metrics and iteration loop
Failure modes and fixes
```

The answer should avoid generic advice like "post consistently" unless it is translated into a concrete cadence, queue, review loop, and recovery path.
