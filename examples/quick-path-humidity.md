# Example: Quick Path

## User request

My room feels humid. Is a dehumidifier actually worth it?

## Expected PavedPath behavior

Use Quick Path unless the user asks for a full buying guide.

```text
Conclusion: If room humidity often stays above about 70%, a real dehumidifier is usually worth it; moisture bags are mostly for closets, not whole rooms.
Why: Mature user experience tends to separate closet-scale fixes from room-scale fixes. Air-conditioner dry mode can help, but it is less consistent for continuous humidity control.
Do this: Measure humidity for two or three days first. If it regularly stays high, choose a compressor dehumidifier sized for the room instead of buying more disposable moisture bags.
```
